# Implementation Logs Compatibility Bridge

**Status:** Compatibility bridge
**Authoritative source:** [`../.implementation/`](../.implementation/)
**Decision:** `DR-013`

`REFERENCE_ARCHITECTURE.md` v1.3 names `DOCS/implementation-logs/` as the implementation log directory. This repository's existing implementation logs remain in `DOCS/.implementation/` under the DR-001 dot-prefix convention until MIG-09 normalizes the log location and naming policy.

Consult `DOCS/.implementation/` for current logs. Do not add implementation logs to this compatibility directory unless a later Decision Register entry makes this path authoritative.
